CSS(Cascading Style Sheets), this is use of decoration page of website.

This is attached from html page in include in head tag.

<!DOCTYPE html>
<html>
<head>
	<title>Cascading style sheet</title>
</head>
<body>
	<style type="text/css">
		.container{
			background-color: red;
			}
	</style>
	<div class="container">
		<h1>This is coding for attache of css for the use for decoration.</h1>
		
	</div>

</body>
</html>

And use for web page decoration. it is help of css include in color, backgroun, image, textposition, height, weidth of any image and videos, text, etc. for the use full in web page decoration.
